# 🧠 Artificial Neural Network (ANN) - Lab Session 5 Learning Module

This repository contains the learning materials, datasets, and Jupyter Notebook implementations for **Lab Session 5: Artificial Neural Networks (ANN)**. It is designed to provide both theoretical understanding and hands-on practical experience in building ANN models for Classification and Regression tasks.

## 🎯 Objectives
* Understand the fundamental concepts and architecture of Artificial Neural Networks.
* Implement ANN for **Classification** tasks (e.g., predicting customer churn).
* Implement ANN for **Regression** tasks (e.g., predicting subscription cancellation metrics).
* Learn how to preprocess data, train models, and evaluate neural network performance.

## 📂 Repository Structure

### 1. 📖 Learning Materials
* **`Modul Mahasiswa Week 7 Artificial Neural Network - 2025.pdf`**: The main guidebook containing the theoretical background, formulas, and step-by-step instructions for this week's module.

### 2. 💻 Code Implementations
* **`Classification_ANN.ipynb`**: A Jupyter Notebook demonstrating how to build and train an ANN model for categorical target variables (Classification).
* **`Regression_ANN.ipynb`**: A Jupyter Notebook demonstrating how to build and train an ANN model for continuous target variables (Regression).

### 3. 📊 Datasets
* **`telecom_churn_dummy.csv`**: A dummy dataset containing telecommunication customer data, used primarily to train the classification model to predict whether a customer will churn.
* **`subscription_cancellation_dummy (1).csv`**: A dummy dataset containing subscription data, utilized for modeling and predicting cancellation trends.

To run the Jupyter Notebooks smoothly, ensure you have Python installed along with the following standard data science and machine learning libraries:

* `pandas` (for data manipulation)
* `numpy` (for numerical operations)
* `scikit-learn` (for data preprocessing and evaluation metrics)
* *Deep Learning Framework* (e.g., `tensorflow`/`keras` or `pytorch` - refer to the notebooks for the specific library used).
